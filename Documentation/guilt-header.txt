guilt-header(1)
===============

NAME
----
guilt-header - Print a patch header

SYNOPSIS
--------
include::usage-guilt-header.txt[]

DESCRIPTION
-----------
Prints either the topmost patches' header or the header of a specified patch.

<patchname>::
	Name of the patch.
