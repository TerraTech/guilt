guilt-prev(1)
=============

NAME
----
guilt-prev - Output name of second topmost applied patch


SYNOPSIS
--------
'guilt-prev' 

DESCRIPTION
-----------
Output name of second topmost applied patch. If guilt-pop(1) is used, this
patch will become the topmost patch.
