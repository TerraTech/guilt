guilt-status(1)
===============

NAME
----
guilt-status - Print the status of files since the last refresh

SYNOPSIS
--------
include::usage-guilt-status.txt[]

DESCRIPTION
-----------
Each file is printed with a prefix that describes it's status.

 A   added
 C   copied
 D   deleted
 M   modified
 R   renamed
 T   type change (e.g., mode)
 U   unmerged
 ?   untracked
