guilt-next(1)
=============

NAME
----
guilt-next - Output the name of next patch to be pushed


SYNOPSIS
--------
include::usage-guilt-next.txt[]

DESCRIPTION
-----------
Output the name of next patch to be pushed. If guilt-push(1) is issued, this
patch would be the topmost patch.
