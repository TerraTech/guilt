guilt-next(1)
=============

NAME
----
guilt-next - Output name of next patch to be pushed


SYNOPSIS
--------
'guilt-next' 

DESCRIPTION
-----------
Output name of next patch to be pushed. If git-push(1) is issued, this patch
would be the topmost patch.
