guilt-push(1)
=============

NAME
----
guilt-push - Push patches onto the tree

SYNOPSIS
--------
include::usage-guilt-push.txt[]

OPTIONS
-------
-f::
	Force the push if the patch doesn't apply cleanly

--all::
-a::
	Push all patches on the tree

<patchname>::
	Push this patch on the tree
